# Macros

Reusable Jinja utilities in `macros/`.

* [pence_to_pounds](pence_to_pounds.md) — converts integer pence to GBP, rounded to 2dp
* [safe_divide](safe_divide.md) — returns null instead of divide-by-zero
* [generate_schema_name](generate_schema_name.md) — overrides dbt's default schema concatenation
